# Getting Started

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/sridhar7601/node-backend.git
2. Navigate into the project directory:

   ```bash
   cd node-backend
3. Install dependencies:(Use either npm or yarn):

   ```bash
   npm install

4. Run the code:
    ```bash
    npm start
